# Decommission Checklist

1. Confirm the target repository and the end state.
2. Update the management ledgers that track active, sleepy, or archived repositories.
3. Record the retirement in the durable event log if the repo is managed there.
4. Preserve only the context that still teaches or supports future audits.
5. Close any related campaign or issue records.
6. Verify that the active pipeline no longer treats the repository as live work.
